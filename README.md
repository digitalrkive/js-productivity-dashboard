# js-productivity-dashboard
A web application designed to help users organise their daily tasks, take quick notes, track time, and personalise their workspace with a theme toggle.
Built using HTML, CSS, and pure JavaScript — no frameworks, no React. This project exists to strengthen my understanding of core JavaScript concepts through building something real and useful.

This project acts as both a practical learning journey and a growing showcase of my JavaScript skills. Each feature is created from scratch using fundamental JavaScript logic.

Core Features (for now)

To-Do List — Add, delete, and complete tasks
Notes Widget — A lightweight notes section for jotting down ideas, reminders, or tasks
Theme Toggle (Dark/Light Mode) — Switch between light and dark themes, implemented with CSS variables and JavaScript
Clock or Timer — A small digital clock and stopwatch to practise setInterval and DOM updates
